## English Version

### 📜 Description
**ItemTimer** is a powerful and highly customizable Minecraft plugin designed to display countdown timers above dropped items and automatically remove them after a set duration. It enhances server performance while providing a visual guide for players.

### ✨ Key Features
*   **⏱️ Floating Timers**: Real-time countdown displayed above items using custom name tags.
*   **🌍 Language Sync**: Automatically translates item names to the player's client language (Translatable components).
*   **📊 Progress Bars**: Visual progress bar showing the remaining time before removal.
*   **💓 Pulsing Effect**: Highlights items with low remaining time (pulsing red color).
*   **⚙️ Deep Customization**:
    *   Set default and custom timers for specific materials (e.g., Diamond, Netherite).
    *   Whitelist/Blacklist specific items or worlds.
    *   Full control over messages and formatting.
*   **🔔 Admin Notifications**: Automatic update checks from GitHub/Modrinth with join notifications for admins.
*   **💾 Auto-Backup**: Creates timestamped JAR backups on every build.

### ⌨️ Commands & Permissions
| Command | Description | Permission |
| :--- | :--- | :--- |
| `/it help` | Show help menu | *None* |
| `/it status` | Show plugin status & tracked items | *None* |
| `/it reload` | Reload configurations | `itemtimer.admin` |
| `/it check` | Manually check for updates | `itemtimer.admin` |

## Phiên bản Tiếng Việt

### 📜 Mô tả
**ItemTimer** là một plugin Minecraft mạnh mẽ và có khả năng tùy biến cao, được thiết kế để hiển thị đồng hồ đếm ngược trên các vật phẩm bị rơi và tự động xóa chúng sau một khoảng thời gian thiết lập. Nó giúp cải thiện hiệu suất máy chủ đồng thời cung cấp thông tin trực quan cho người chơi.

### ✨ Các Tính năng Chính
*   **⏱️ Đồng hồ nổi**: Đếm ngược thời gian thực hiển thị phía trên vật phẩm bằng thẻ tên tùy chỉnh.
*   **🌍 Đồng bộ ngôn ngữ**: Tự động dịch tên vật phẩm sang ngôn ngữ máy khách của người chơi (Translatable components).
*   **📊 Thanh tiến trình**: Hiển thị thanh tiến trình trực quan cho biết thời gian còn lại trước khi biến mất.
*   **💓 Hiệu ứng nhấp nháy**: Làm nổi bật các vật phẩm sắp bị xóa (màu đỏ nhấp nháy).
*   **⚙️ Tùy biến sâu**:
    *   Thiết lập thời gian mặc định và riêng biệt cho từng loại vật phẩm (ví dụ: Kim cương, Netherite).
    *   Danh sách trắng/đen cho từng vật phẩm hoặc thế giới cụ thể.
    *   Toàn quyền kiểm soát tin nhắn và định dạng hiển thị.
*   **🔔 Thông báo cho Admin**: Tự động kiểm tra cập nhật từ GitHub/Modrinth và thông báo khi Admin vào server.
*   **💾 Sao lưu tự động**: Tạo bản sao lưu JAR kèm mốc thời gian sau mỗi lần build.
